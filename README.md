# Stacks Maritime Trading

## Mission
A decentralized platform for maritime logistics and trade automation.

## Tech Stack
- Stacks Blockchain (Clarity 4, Epoch 3.3)
- @stacks/connect for wallet interaction
- @stacks/transactions for on-chain logic
- WalletConnect for multi-wallet support
- Hiro Chainhooks for real-time indexing

## Features
- Vessel registration and tracking
